# 项目简介
本项目是一款基于C语言和easys库开发的经典贪吃蛇小游戏，项目历经多个版本迭代，不断优化视觉效果和功能。

## 开发历史
1.0 版本：采用控制台界面，实现基础的贪吃蛇功能，逻辑简单但功能完整。

2.0 版本：引入 EasyX 图形库，重新编写显示逻辑，实现更直观的图形化界面。

3.0 版本：时间有限，仅完成了以下功能：
- 增加了主页面设计。
- 显示历史最高分。
- 导入并使用图形素材，提升整体视觉效果。
